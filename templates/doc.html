Email Address: required.

Gene Information: Input 'DNA seq' from 5' to 3' end. Length Range from 0.1 to 100 kbp. The 'DNA seq' is required and the other items is optional.

Parameters For Gene Synthesis: Parameters for real time PCR assembling.

Overlap Length: Users can tune this parameter to change the length of overlap between oligos and output different oligonucleotide sets. Default parameter minimum 20bp and maximum 30bp will produce oligo overlap length range from 16bp to 34bp.

Result Type: Users can select 'Gapless' or 'Gap' oligo set for PCR assembly. When selecting 'Gap', the outputted oligonucleotide sets with gap between consecutive oligonucleotides possess more uniform melting temperature. The outputted gapless oligonucleotide sets also can be used for LCR-based gene synthesis. Furthermore, users can slect 'another result' option in the output 'Result' tab to get alternative oligo set, which is segmented from 3' end of input sequence. In some circumstances, a small tail may be added to the given sequence automatically by the program. The tail will be highlighted in red in the final result. It can be eliminated in PCR reaction by using the given primers.

Long DNA Assembly(optional): Sequence will be partitioned into # of pools of oligodeoxynucleotides with homologous melting temperature for long sequence. The last oligodeoxynucleotide in the previous pool has overlap region range from 16bp to 34bp with the first oligodeoxynucleotide in the next pool.

Is Verification Required: Analyse the mismatches between oligonucleotides based on thermodynamic processes and provide the concentration of potential mismatches. It will analyse the potential mismatches between any oligonucleotide couples and present the potential mismatches and their concentration. If 'Yes' option is slected, users can determine the temperature, concentration of oligos for mismatch analysis.
